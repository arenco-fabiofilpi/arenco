# Changelog

Todas as mudanças importantes deste projeto serão documentadas aqui.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/), e esse projeto segue o versionamento [SemVer](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2025-06-08

### Adicionado

- Estrutura inicial do monorepo para o sistema da construtora.
- Módulos organizados em: `backend/`, `frontend/`, `mobile/`, `scripts/`, `deploy/`.
- Script `build-all.sh` para build de todos os serviços.
- Script `start-backend.sh` como exemplo de deploy com `nohup`.
- `version.txt` para controle da versão unificada do pacote de negócio.
- `.gitignore` cobrindo Maven, Node.js, Flutter e IDEs.
- `README.md` explicando como usar e navegar no projeto.
